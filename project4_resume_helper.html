구현 방식

@input_guardrail 데코레이터
OpenAI Agents SDK의 입력 가드레일을 활용. 에이전트 실행 전 tripwire 조건 확인 후 자동 차단.
Pydantic 스키마 검증
ResumeGuardrailOutput 모델로 is_harmful 필드를 구조화. 타입 안전성 확보.
각 에이전트 System Prompt
허위 경력 작성 요청, 시스템 지시 공개 요청을 에이전트 레벨에서도 거절하도록 명시.
Dev Log

Day 1
